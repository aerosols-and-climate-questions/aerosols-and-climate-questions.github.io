About this project

Each page has questions about a different topic in aerosols and their interaction with the climate and corresponds to a chapter in Aerosols and Climate. There are three types of questions:

  • Review questions test core knowledge and are aimed at students who have read the corresponding book chapter.
  • Application questions ask you to apply concepts to everyday questions in aerosol science. These are aimed at students developing understanding of the subject and are approximately at the level of a first-year PhD candidate.
  • Mastery questions require readers to think deeply about the science and apply one or more concepts to a real challenge in research. They are deep and akin to the level of a question asked in a PhD viva. Some go beyond the scope of the chapter's contents.

We welcome contributions from the community. The questions are open-source and stored on our Github repository. If you want to contribute a question, you can create a pull request and we'll review it. Similarly, if you've found an error in a question, submit an issue on our Github repo to report it.
